## Post 4: Technical Deep Dive

**Title:** "How I Built an AI-Controlled Desktop Avatar"

**Content:**
```
Tech stack breakdown! 🔧

Frontend:
• Electron - Transparent window app
• Three.js - 3D WebGL rendering
• GLTFLoader - GLB file support

Backend (Bridge):
• WebSocket server (ws library)
• IPC for Electron ↔ Renderer
• Port 8765 for AI connection

AI Integration:
• OpenClaw sends WebSocket commands
• Real-time animation control
• Expression/mood changes
• Skeleton manipulation possible

Challenges solved:
✅ Transparent background (not easy in Electron!)
✅ Click-through (window but not blocking)
✅ Always on top (stays visible)
✅ Smooth animations (60fps)

Next: Adding voice lip-sync and webcam eye tracking! 👀
```
