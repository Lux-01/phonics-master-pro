## What Gets Installed?

1. **WSL2** - Linux on Windows
2. **Ubuntu** - Operating system
3. **Ollama** - AI model server
4. **Kimi-K2.5** - The AI brain (~4GB download)
5. **OpenClaw** - Your personal AI gateway

## Requirements

- Windows 10 version 2004+ or Windows 11
- Administrator access
- 8GB RAM minimum
- 10GB free space

## Help

If the automatic installer fails:
1. Check `C:\Users\YOURNAME\AppData\Local\Temp\openclaw-install.log`
2. Follow the manual steps in `INSTALLATION-GUIDE.md`
