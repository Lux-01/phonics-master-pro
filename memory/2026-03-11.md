### 2. Bugs Found & Fixed

**Bug #1 - CRITICAL: Path Conversion**
- **Issue:** Windows paths like `C:\Users\Name\file` passed to WSL
- **Effect:** Backslashes interpreted as escape characters → fails
- **Fix:** Added `ConvertTo-WSLPath()` function to convert to `/mnt/c/path` format

**Bug #2 - CRITICAL: Interactive WSL Hang**
- **Issue:** `wsl --install` prompts for username/password
- **Effect:** Script appears to freeze (waits for input forever)
- **Fix:** Added pre-checks, detects first-time setup, instructs restart

**Additional Fixes:**
- Added pre-flight checks (admin, Windows version, disk space)
- Added logging system
- Simplified nested heredocs
- Added better error handling
