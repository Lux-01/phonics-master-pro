# Task Templates

## Template 1: Research Task

```markdown
## Task: Research [Topic]
**Mode:** Autonomous  
**Checkpoints:** None (low risk)

### Steps
1. Search for authoritative sources
2. Read top 3-5 sources
3. Extract key findings
4. Compare sources (note conflicts)
5. Synthesize into summary
6. Deliver findings

### Deliverables
- Research summary
- Source list with quality scores
- Key findings
- Recommendations
```

## Template 2: Code Implementation

```markdown
## Task: Implement [Feature]
**Mode:** Autonomous with checkpoints  
**Checkpoints:** After architecture, after MVP

### Steps
1. Research API/libraries needed
2. Design architecture
3. [CHECKPOINT] Present design
4. Implement core functionality
5. Add error handling
6. Write tests
7. [CHECKPOINT] Present MVP
8. Refine based on feedback
9. Finalize and document

### Deliverables
- Implementation code
- Test cases
- Documentation
- Usage examples
```

## Template 3: Debugging

```markdown
## Task: Debug [Issue]
**Mode:** Autonomous  
**Checkpoints:** If complex fix needed

### Steps
1. Read relevant code
2. Identify reproduction steps
3. Add logging/debugging
4. Run with test data
5. Identify root cause
6. Implement fix
7. Test fix
8. Verify no regressions
9. Document fix

### Deliverables
- Root cause analysis
- Fix description
- Testing results
```

## Template 4: Refactoring

```markdown
## Task: Refactor [Codebase]
**Mode:** Checkpoint every file group  
**Checkpoints:** Yes (high risk)

### Steps
1. Analyze current code
2. Identify improvement areas
3. Plan refactoring approach
4. [CHECKPOINT] Get approval
5. Execute refactoring
6. Run tests
7. Update documentation
8. Log as EVO entry

### Deliverables
- Refactored code
- Test results
- Documentation updates
- EVO log entry
```

## Template 5: Organization

```markdown
## Task: Organize [Workspace/Area]
**Mode:** Shadow first, then execute  
**Checkpoints:** Before any moves

### Steps
1. Analyze current structure
2. Identify problems
3. Propose new structure
4. [CHECKPOINT] Get approval
5. Create backups
6. Execute reorganization
7. Update references
8. Verify integrity

### Deliverables
- New structure
- Backup locations
- Updated references
```
